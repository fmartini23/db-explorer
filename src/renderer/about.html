DB Explorer

Version 1.0.0

About

DB Explorer is a powerful database management tool that allows you to connect to multiple database systems, execute queries, and manage your database objects with an intuitive user interface.

Features

  • Multi-database support (MySQL, PostgreSQL, SQL Server, SQLite, Oracle, MongoDB)
  • Intuitive object explorer for browsing database objects
  • Advanced SQL editor with syntax highlighting
  • Query execution with result visualization
  • Table design and creation tools
  • Import and export functionality
  • Connection management
  • Dark and light theme support
